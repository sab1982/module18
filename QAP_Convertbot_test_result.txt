Ошибки конвертации:
- команда: доллар рубль 1, результат: Что-то пошло не так с 'ЮэСДэ'
- команда: рубль доллар 1, результат: Что-то пошло не так с 'RUB'
- команда: евро доллар 1, результат: Что-то пошло не так с 'EUR'
- команда: доллар евро 1, результат: Что-то пошло не так с 'ЮэСДэ'

Ошибки значений при конвертации:
- значения стоимости валют перепутаны местами (Переводим рубль в евро 1 рубль = 66.65 евро).
- невозможно посчитать количество валюты больше, чем 1 

Успешные кейсы только для валидации ошибок пользователя:
- количество вводимых аргументов = 3, результат - Введите команду или 3 параметра
- проверка цифрового значения третьего элемента, результат - Не смог обработать количество десять
- проверка ввода одинаковых валют, результат - Нельзя перевести одинаковые валюты евро (проверенно на всех доступных валютах)
- проверка корректности ввода названия валюты первого и второго аргумента, результат - Не смог обработать валюту рабль (команда: евро рабль 1, рабль евро 1)
